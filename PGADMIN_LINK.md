## First Time Setup

1. **When pgAdmin opens**, you'll be asked to set a master password
   - This is for pgAdmin itself (not PostgreSQL)
   - Choose any password you'll remember

2. **After setting password**, you'll see the pgAdmin interface

3. **Connect to PostgreSQL:**
   - Right-click **"Servers"** → **"Create"** → **"Server"**
   
   **General Tab:**
   - Name: `Local PostgreSQL`
   
   **Connection Tab:**
   - Host: `localhost`
   - Port: `5432`
   - Username: `yigitilseven`
   - Password: (leave empty)
   - Database: `postgres`
   
   - Click **"Save"**

4. **View your database:**
   - Expand **"Servers"** → **"Local PostgreSQL"**
   - Expand **"Databases"** → **"stratejiplus"**
   - Expand **"Schemas"** → **"public"** → **"Tables"**
   - You'll see all 13 tables!
